PHASE 4 - WRITE TESTS (Iterative)

```python
def test_should_handle_success_case():

    # Then
    assert result == "expected"
```

2. Use fixtures:
```python
@pytest.fixture
def service():
    return MyService()

def test_with_fixture(service):

    assert result == "expected"
```

```python
def test_calls_repository(mocker):

```python
